# Code of Conduct

This repository aims to stay respectful, practical, and welcoming.

## Expected behavior

- be constructive and specific
- respect different experience levels
- focus feedback on the work, not the person
- help keep the project clear and useful

## Unacceptable behavior

- harassment or personal attacks
- spam, trolling, or bad-faith review comments
- sharing private information
- hostile or discriminatory behavior

## Enforcement

Project moderation decisions are made by the repository owner. If a contribution or discussion is clearly harmful, it may be edited, locked, or removed.
