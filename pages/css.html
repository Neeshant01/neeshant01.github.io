Project 2: CSS Button Sequence Game

Four bright buttons arranged in a square, each with a different animation, hover glow, and active press behavior. A CSS counter numbers the buttons automatically.

Project 3: CSS Interactive Snake Path Game

A visual snake-style path demo using a CSS grid background, animated movement, color gradients, and a glowing food target.

Project 4: CSS 3D Interactive Cube

A pure CSS cube with six faces, 3D transforms, perspective, automatic load animation, and a hover rotation interaction.

Key CSS Concepts Demonstrated

Grid and Flexbox: used for card layouts, section organization, and button placement.

Transforms: 3D rotateY flips, perspective cube faces, hover scale, and animated movement.

Animations and Transitions: pulsing buttons, moving snake path, cube rotation, and shape sequences.

Variables and Media Queries: shared colors, card sizing, and responsive layout updates.
