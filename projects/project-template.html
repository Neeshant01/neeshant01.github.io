How to add a new project

Copy this file, rename it, and replace the content with your real project details. Then add one new item in projects/projects.json so the homepage can show the new card.

Example JSON item

Add a new object inside projects/projects.json with a title, type, summary, stack, and link to your project page.

Example link value: projects/my-new-project.html

Recommended structure

  • Short project title
  • One clear summary
  • Tech or tools used
  • What was built
  • What was learned
